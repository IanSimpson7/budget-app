# Summary: Bump GitHub Actions tags to latest majors (Node 24)

## What changed

`.github/workflows/deploy.yml` — five action tags bumped to current latest
majors (verified live via `gh api repos/<action>/releases/latest` on 2026-05-30):

| Action | Old | New |
|--------|-----|-----|
| actions/checkout | v4 | v6 |
| actions/setup-node | v4 | v6 |
| actions/configure-pages | v4 | v6 |

## Why

GitHub force-migrates Actions runners to Node 24 on 2026-06-02; the old
`@v4`/`@v3` majors ran on Node 20 and would fail after migration. Closes the
STATE.md dated follow-up [ACT BEFORE 2026-06-02].
