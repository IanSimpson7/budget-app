# Quick Task 260530-81h: Bump GitHub Actions tags to latest majors (Node 24)

## Why

GitHub force-migrates Actions runners to Node 24 on **2026-06-02**. The pinned
`@v4`/`@v3` Pages actions ran on Node 20 and would break (or emit deprecation
failures) after that date. This is the dated follow-up tracked in STATE.md.

## Task

Bump the five action tags in `.github/workflows/deploy.yml` to their current
latest majors (verified live via the GitHub releases API on 2026-05-30):

| Action | Old | New |
|--------|-----|-----|
| actions/checkout | v4 | v6 |
| actions/setup-node | v4 | v6 |
| actions/configure-pages | v4 | v6 |
| actions/upload-pages-artifact | v3 | v5 |
| actions/deploy-pages | v4 | v5 |

- **files:** `.github/workflows/deploy.yml`
- **action:** Replace the five `@vN` tags with the latest majors above.
- **verify:** `git diff` shows exactly five tag changes; YAML structure unchanged;
  the next push to `main` deploys green on the Actions tab.
- **done:** Workflow references only Node 24-compatible action majors; STATE.md
  dated follow-up closed.
